Chess-CheckMate-Detector ♟️
A C++ engine built to master the board! It calculates complex checkmate sequences by simulating every possible move. 🧠

What it does:
Checkmate Detector: Pinpoints if the King is trapped. 👑

Mate in 1: Finds instant game-ending moves. ⚡

Mate in 2: Executes a deep search to force a win in two moves. 🔍

Board Logic: Validates legal moves, pins, and turn-based constraints. 🛡️

How it works:
It uses a recursive simulation approach to evaluate the entire decision tree, ensuring that every opponent's response is accounted for before confirming a forced mate. 📈
